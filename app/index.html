Kubernetes + MySQL Demo

สวัสดี 👋
นี่คือ WebApp เชื่อมต่อฐานข้อมูล

หน้าเว็บนี้ใช้สำหรับทดสอบการส่งข้อมูลจาก Web Application ไปยัง MySQL ที่รันอยู่บน Kubernetes โดยสามารถนำไปต่อยอดกับ Ingress, ArgoCD, Longhorn และ CI/CD ได้ต่อเลย

  • ✅ Frontend HTML เรียบง่ายและพร้อมใช้งาน
  • ✅ Backend เชื่อมต่อ MySQL ผ่าน PHP
  • ✅ เหมาะสำหรับ Lab, Demo และฝึก CI/CD
  • ✅ ต่อกับ StatefulSet + Persistent Storage ได้
